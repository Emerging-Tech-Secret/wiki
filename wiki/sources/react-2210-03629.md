# ReAct: Synergizing Reasoning and Acting in Language Models

**Source:** Shunyu Yao, Jeffrey Zhao, Dian Yu, Nan Du, Izhak Shafran, Karthik Narasimhan, Yuan Cao.
*ReAct: Synergizing Reasoning and Acting in Language Models.* arXiv:2210.03629 [ref_1].
**Trust tier:** A (arXiv). **Snapshot:** `raw/arxiv_2210_03629.md` (fetched 2026-06-10).

## What it says
- Reasoning (e.g. chain-of-thought) and acting (e.g. action-plan generation) in LLMs had mostly been
  studied as separate topics; ReAct combines them by generating reasoning traces and task-specific
  actions in an **interleaved** manner [ref_1].
- **Reasoning traces** help the model induce, track, and update action plans and handle exceptions;
  **actions** let it interface with external sources (knowledge bases, environments) to gather
  additional information [ref_1].
- On **HotpotQA** (question answering) and **Fever** (fact verification), ReAct queries a simple
  Wikipedia API and reduces the hallucination and error propagation seen in chain-of-thought-only
  reasoning [ref_1].
- On **ALFWorld** and **WebShop** (interactive decision making), ReAct beats imitation- and
  reinforcement-learning baselines by an **absolute success rate of 34% and 10%** respectively, using
  only **one or two in-context examples** [ref_1].
- ReAct improves **human interpretability and trustworthiness** versus methods lacking reasoning or
  acting components [ref_1].
